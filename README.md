# 🌸 FourSeasons - 四季特效插件 for Typecho 1.3.0

为 Typecho 博客添加四季粒子特效，支持春/夏/秋/冬四种场景独立开关控制，并支持简约模式切换。

## ✨ 功能特性

- **🌸 春季 - 樱花飘落**：单瓣粉色樱花从顶部飘落，带 3D 翻转效果（简约模式为粉色圆点）
- **🌧️ 夏季 - 细雨飘洒**：斜向飘落的雨丝，营造夏日清凉氛围（简约模式为蓝色短竖线）
- **🍂 秋季 - 落叶飘零**： realistic 枫叶形状从顶部旋转飘落（简约模式为橙色圆点）
- **❄️ 冬季 - 雪花纷飞**：真实圆润六角雪花，中心圆盘+六个圆角突起（简约模式为白色圆点）
- **🎨 简约模式**：后台一键切换，所有季节变为圆点/简单形状，性能更好更简洁
- **🔧 独立控制**：每个季节可独立开关，自定义粒子数量（密度）

## 📸 预览

- **正常模式**：真实形状（樱花、雨丝、枫叶、六角雪花）
- **简约模式**：统一圆点（粉、蓝、橙、白），大小一致，仅颜色区分季节

## 📦 安装方法

1. 创建文件夹 `FourSeasons`，在里面创建 `assets` 子文件夹
2. 将 `Plugin.php` 放到 `FourSeasons/` 目录
3. 将 `seasons.js` 放到 `FourSeasons/assets/` 目录
4. 将整个 `FourSeasons` 文件夹上传到 Typecho 的 `/usr/plugins/` 目录
5. 登录后台，进入"控制台" → "插件管理"找到 "FourSeasons" 点击启用
6. 点击"设置"选择要开启的季节效果或切换简约模式


## ⚙️ 配置说明

| 设置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| 🎨 简约模式 | 开关 | 关闭 | 开启后所有季节变为圆点形状 |
| 🌸 春季 | 开关 | 关闭 | 樱花飘落效果 |
| 樱花数量 | 数字 | 40 | 范围 10-150 |
| 🌧️ 夏季 | 开关 | 关闭 | 细雨飘洒效果 |
| 雨丝数量 | 数字 | 60 | 范围 20-200 |
| 🍂 秋季 | 开关 | 关闭 | 落叶飘零效果 |
| 落叶数量 | 数字 | 35 | 范围 10-100 |
| ❄️ 冬季 | 开关 | 开启 | 雪花纷飞效果 |
| 雪花数量 | 数字 | 50 | 范围 10-200 |

## 💡 使用建议

- **建议只开启一个季节效果**，多个同时开启可能影响页面性能
- **简约模式**适合追求简洁或性能敏感的场景，圆点大小统一（春秋冬相同）
- **冬季效果**适合全年使用，雪花小巧圆润（2-4px），不会太抢眼
- 移动端建议开启简约模式或降低粒子数量

## 🔧 技术细节

- 基于 HTML5 Canvas 渲染
- 使用 `requestAnimationFrame` 实现流畅动画
- 响应式设计，自动适应窗口大小变化
- 零依赖，无需引入外部库
- 兼容 Typecho 1.3.0 + PHP 7.2+
